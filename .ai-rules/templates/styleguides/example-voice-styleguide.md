# Example Voice Styleguide

This is a public example styleguide. Replace it with your own private
styleguide if desired.

## General Style

- Be direct.
- Prefer concrete language.
- Avoid filler.
- Use humor sparingly.
- Prefer useful specificity over broad encouragement.

## Avoid

- Generic corporate tone
- Overly long intros
- Unnecessary apologies
- Fake enthusiasm

## Notes for the agent

- If a private voice styleguide is available at the path configured in
  `.ai-local/styleguides.yaml`, prefer it over this generic example.
- This file is intentionally generic — do not infer specific voice
  preferences from it beyond the principles listed above.
